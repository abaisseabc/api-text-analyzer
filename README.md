# api-text-analyzer

API для обнаружения ИИ в тексте

## Структура

- blueprints - все компоненты приложений:
  - analyzer - логика + API
- app.py - главный файл приложения

## Запуск приложений 

Установка зависимостей:
```shell
pip install -r requirements.txt
```

Запуск:
```shell
python3 app.py
```
